# Dashboard Interativo - Airbnb Rio de Janeiro

## Sobre o Projeto
Dashboard profissional em Power BI para análise do mercado de hospedagem do Airbnb no Rio de Janeiro, com visualizações interativas e filtros dinâmicos.

## Objetivos
- Fornecer insights visuais sobre o mercado Airbnb Rio
- Permitir análise segmentada por bairro, tipo de imóvel e preço
- Criar ferramenta interativa para tomada de decisão
- Desenvolver habilidades em Business Intelligence e Power BI

## Tecnologias Utilizadas
- **Power BI** - Criação de dashboards e visualizações
- **Business Intelligence** - Análise estratégica de dados
- **Data Visualization** - Apresentação visual de insights

## Dataset
**Fonte:** Análise prévia do Airbnb Rio ([ver projeto completo](https://github.com/Matheus-Diniz-Silva/analise-airbnb-rio))

**Dados incluídos:**
- 2.000 imóveis representativos
- Informações de localização, preço e avaliações
- Segmentação por bairro e tipo de imóvel
- Categorização de preços
